# News Explorer

Aplicação mobile com o intuito de mostrar as principais notícias da atualidade, utilizando a API pública do NewsAPI(https://newsapi.org/).

## Tecnologias

1. React Native
2. TypeScript

## Como executar

1. Instalando as dependências

   ```bash
   npm install
   ```

2. Inicializando o app

   ```bash
    npx expo start
   ```
